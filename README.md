# Nebo Rangeslider

Angular JS wrapper for RangesliderJS

## Installation

```
npm i nebo-rangeslider --save
```

```
bower i nebo-rageslider --save
```


## Usage

```js
angular.module('app', [
  'rangeslider'
])
```

or

```
angular.module('app', [
  require('nebo-rangeslider')
]);
```
